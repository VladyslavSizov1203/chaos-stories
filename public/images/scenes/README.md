# Scene Images

This directory contains background images for game scenes.

## Image Requirements
- Format: JPEG (good compression, wide support)
- Dimensions: 1920x1080 (16:9 landscape)
- File size: < 300KB per image
- Naming: kebab-case (e.g., tavern-entrance.jpg)

## MVP Placeholder
For MVP development, we're using CSS gradients as placeholders.
Actual scene artwork will be added in a future sprint.
